# auto-configure

## How use it

```sh
$ su -
$ apt install git
$ git clone https://github.com/theo-grivel/42-born2beroot
$ cd 42-born2beroot/autoconfig
# set the variable in run.sh file
$ chmod +x run.sh
$ ./run.sh
```
